# ?? Sistema de Gestão de Lotes para Produção de Ovos

Este projeto é uma API REST desenvolvida com Spring Boot para auxiliar produtores rurais na gestão de lotes de aves poedeiras. O sistema oferece controle de postura, perdas, vacinação e relatórios analíticos.

## ?? Funcionalidades
- Cadastro e gerenciamento de lotes
- Cálculo de perdas e desempenho
- Controle de vacinação com agendamento
- Alertas automáticos via mensageria
- Relatórios analíticos por lote

## ??? Tecnologias utilizadas
- Java 17
- Spring Boot
- Spring Data JPA
- Swagger (OpenAPI)
- Hibernate
- IntelliJ IDEA
- PostgreSQL / H2
- Scheduled Tasks
